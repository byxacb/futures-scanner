# 主策略：受控激进趋势跟踪（Master Playbook v1.0）

> 这份文档是本项目的"宪法"。所有日常交易决定都必须能在这里找到依据。
> 与 `rules/评分公式拆解.md`、`primer/03_止损与仓位管理.md`、`books/_consensus.md` 互为支撑。

---

## 一、核心信念（不可妥协）

1. **评分公式 60% 权重在惩罚风险**——我们玩的不是收益率最大化，而是综合得分最大化
2. **"前 4 概率"最大化 ≠ 收益率最大化**——前 4 = 综合得分前 4，而综合得分 = 收益率×40% + (1-回撤)×20% + 夏普×30% + (1-波动率)×10%
3. **趋势是朋友，但趋势不是常态**——一年里真正的大趋势 5-10 次，其余时间是噪音
4. **仓位是唯一不可逆变量**——技术指标错了可以改，仓位过大爆仓就回不来了
5. **截短亏损让利润奔跑**——不是口号，是数学（见下方）

数学证明 5：胜率 40%、盈亏比 3:1 的策略
- 期望 R = 0.4×3 + 0.6×(-1) = +0.6（每笔交易期望赚 0.6 个 R）
- 如果改成胜率 60%、盈亏比 0.5:1（短利落袋）：
- 期望 R = 0.6×0.5 + 0.4×(-1) = -0.1（负期望！）

胜率高不等于赚钱——盈亏比才是。

## 二、品种池（3-5 个，低相关）

### 默认品种池（首发）

| 品种 | 代码 | 板块 | 选择理由 |
|------|------|------|---------|
| 螺纹钢 | rb | 黑色 | 流动性最好、A 股周期联动、政策驱动明显 |
| 豆粕 | m | 农产品 | 美豆+国内库存双驱动、与黑色相关性低 |
| 沪金 | au | 贵金属 | 宏观对冲、与商品板块负相关 |
| 沪深 300 IF | IF | 金融 | A 股大盘、流动性极佳 |
| 原油 | sc | 能化 | 全球宏观、与其他品种相关性最低 |

为什么是这 5 个：
- **板块覆盖**：黑色/农产品/贵金属/金融/能化——5 大主流板块各 1 个
- **相关性**：5 个品种两两相关性都低（<0.5）
- **流动性**：日均成交量都在前 10
- **基本面易追踪**：每个品种关键基本面数据都有公开来源

### 品种池调整规则

每周日复盘后评估是否换品种：
- 移除：最近 4 周 ATR 突破方向连续 3 次反向（信号失灵）
- 移除：成交量持续低于全市场前 20（流动性变差）
- 增加：候补品种（见下）

### 候补品种（次优先）

铁矿 i、棕榈 p、沪铜 cu、沪铝 al、PTA、玻璃 FG、IH、IC、TS、TF

⚠️ **绝对不碰**（流动性差或剧烈难控）：纯碱 SA、苹果 AP、红枣 CJ、纸浆 SP、各种期权（除非有专门策略）

## 三、入场信号（多重过滤）

### 主信号：通道突破

**System 1（短）**：日线突破 20 日新高（多）/ 新低（空）
**System 2（长）**：日线突破 55 日新高 / 新低

System 1 信号更频繁但假突破多；System 2 信号少但确信度高。

### 过滤条件（必须全部满足才入场）

| 条件 | 要求 | 工具检测 |
|------|------|---------|
| A. 通道突破 | 日线收盘价高于 20 日最高（多） | indicators/compute.py |
| B. 趋势方向 | 日线 MA20 > MA60（多）或 < （空） | indicators |
| C. 成交量 | 突破当日成交量 ≥ 20 日均量 × 1.2 | indicators |
| D. 持仓量 | 突破当日 OI 增加（说明主力开仓不是平仓） | data |
| E. 板块联动 | 同板块至少 1 个相关品种同方向 | indicators |
| F. 重大事件 | 未来 24h 内无 FOMC/CPI/非农/重要会议 | briefing |

A+B 是底线，C/D/E 至少满足 2 个，F 是必须。

### 高确信信号（可加注 1.5-2 倍仓位）

A+B+C+D+E 全部满足 + 基本面也支持（如库存/政策）→ "高确信"——基础仓 30% → 加到 60%

### 假突破识别（不入场）

- 突破当日成交量未放大
- 突破后 30 分钟内回到突破位以下
- 板块内无任何品种同步

## 四、仓位规则（这是核心 KPI）

### 单笔仓位（基础公式）

```
最大手数 = (账户净值 × 1%) / (2 × ATR × 合约乘数)
```

**单笔风险 = 1% 而不是 2%**——比 primer 里讲的更严格一档。原因：
- 海龟原版用 2% 的人不少都吃过大亏
- 本次比赛百日级别，要求生存率高
- 1% 给我们 100 单容错（实际不会用到）

### 单品种保证金占用上限：10%

100 万账户 → 单品种最多用 10 万保证金。

### 加仓阶梯（仅在高确信信号 + 浮盈 1×ATR 之后）

```
第 1 单位（初始）：达到入场信号即开
第 2 单位：浮盈 +1×ATR 且未跌破 1×ATR 移动止损
第 3 单位：浮盈 +2×ATR 且未跌破 2×ATR 移动止损
第 4 单位：浮盈 +3×ATR 且趋势加速（MACD 柱继续放大）
```

每加一次仓，所有止损同步上移到"-2×ATR from latest entry"。

最多 4 个单位（海龟原则），单品种总保证金不超过 10%。

### 总仓位上限

| 维度 | 上限 |
|------|------|
| 单方向（多或空合计） | 40% 保证金 |
| 总保证金占用 | 70% |
| 留白（应对追保+黑天鹅+加注子弹） | ≥30% |

### 仓位调整事件

- 账户回撤 >3% → 单笔风险降到 0.5%（仓位减半）
- 账户回撤 >6% → 全平 + 一周休战
- 账户净值新高 → 仍按 1% 风险（不是 1.2%），避免膨胀
- 重大事件前 24h → 现有头寸减半 / 不开新单

## 五、止损规则（铁律）

### 入场即挂止损单（绝不"心里止损"）

止损价 = 入场价 ± 2×ATR（多头减，空头加）

### 移动止损（向有利方向 only）

```
价格每涨 1×ATR → 止损上移 1×ATR
最低保护：止损不低于成本价（一旦浮盈 >2×ATR）
```

### 时间止损

- 持仓 5 个交易日仍未浮盈 1×ATR → 平仓（信号没走出来 = 信号无效）

### 紧急止损

- 持仓品种当日出现 >5% 单边行情（与你方向反向）→ 立即手动减半，剩余按 ATR 止损
- 出现穿仓苗头（保证金占用 >85%） → 立即减仓到 60% 以下

## 六、止盈规则

### 主止盈：移动止盈（trailing stop）

不主动平多/空——靠移动止损被动出场。原因：截断亏损让利润奔跑。

### 部分止盈（在特定情况）

- 浮盈 >3×ATR 且周线 RSI >75 → 平 1/3 仓位
- 浮盈 >5×ATR 且日线出现反转 K 线（吞没、黄昏星）→ 平 1/2 仓位
- 重大事件前 24h → 平 1/3 仓位

### 不止盈的情况

- 浮盈 <2×ATR → 不动（让趋势继续）
- 周末持仓且无重大事件 → 不动

## 八、心态与执行铁律

1. **不预测，只反应**——信号触发就执行，不要等"再确认一下"
2. **进场前默念**："如果说错了我会在哪平仓？最大损失多少？我能承受吗？"（Kovner 三问）
3. **赚钱时**：别加仓兴奋单，按金字塔阶梯加
4. **亏钱时**：别加仓"摊薄"（这是马丁，禁止）
5. **冲动时**：离开屏幕 30 分钟
6. **复盘时**：诚实，每笔单写下"是策略错还是执行错"

## 九、周复盘 + 评分公式自估

每周日：

1. **计算本周**：总收益率、最大回撤、夏普比率（周收益均值/周收益标准差×√52）、波动率（周标准差）
2. **估算自己排名**（对标 N=1000 假设）：
   - 收益率分位（前 X%）
   - 回撤分位
   - 夏普分位
   - 波动率分位
3. **代入公式**：S = 0.4×R + 0.2×DD + 0.3×Sharpe + 0.1×Vol
4. **对比官方**：等比赛开始后用官方排名校准
5. **调整**：哪一项最弱？下周针对性优化

## 附录：决策树（每次开仓前用）

```
1. 当前是趋势市还是震荡市？（看 ADX）
   - 震荡市 → 不开新单（除非套利）
   - 趋势市 → 进入步骤 2

2. 有突破信号吗？（A+B 是否满足）
   - 否 → 等
   - 是 → 进入步骤 3

3. 过滤条件 C/D/E 至少 2 个满足？
   - 否 → 不开
   - 是 → 进入步骤 4

4. 未来 24h 有重大事件吗？（F）
   - 是 → 不开
   - 否 → 进入步骤 5

5. 按公式算手数
   - 单笔最大亏损 = 账户 ×1%
   - 止损距离 = 2×ATR
   - 最大手数 = 上面 / (止损距离 × 合约乘数)

6. 加注？满足"高确信"5 条全有 + 基本面也支持？
   - 是 → 仓位 ×1.5
   - 否 → 标准 1 单位

7. 单品种保证金 ≤10%？总保证金 ≤70%？单方向 ≤40%？
   - 否 → 减手数到满足
   - 是 → 进入步骤 8

8. 同步挂止损单 + 设定加仓触发位 + 记录到 daily/

9. ⚠️ 进场前 10 秒：Kovner 三问
   - 错了我在哪平？
   - 最大损失多少？
   - 我能承受吗？

10. 执行。然后管住手，让规则替你工作。
```
