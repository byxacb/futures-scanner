# 止损规则手册（不容妥协的部分）

> 与 `master_playbook.md` 第五节配套。

---

## 第一原则：止损单必须实盘挂

**心里止损 = 没止损**。所有头寸开仓后 30 秒内必须在交易软件挂止损单。

工具会在 daily 简报和 monitor 实时通知里检查"未挂止损单"的头寸，并强提示。

## 5 种止损（按触发频率）

### 1. ATR 硬止损（最常用）

- 入场即挂
- 多头：止损价 = 入场价 - 2×ATR
- 空头：止损价 = 入场价 + 2×ATR
- 触发即市价平仓，不犹豫

### 2. 移动止损（trailing stop）

- 浮盈每增加 1×ATR → 止损同向移动 1×ATR
- 浮盈达 2×ATR 后：止损至少在成本价（不再让赚的回头变亏）
- 浮盈达 3×ATR 后：止损至少在 1×ATR 利润处

实操要点：移动止损用**条件单**或交易软件的"跟踪止损"功能，不要手动每天调。

### 3. 时间止损

- 持仓 5 个交易日仍未浮盈 ≥1×ATR → 平仓
- 逻辑：信号失效。市场没给你预期的反应 = 信号 invalid

### 4. 事件止损（防黑天鹅）

- 重大事件前 24h：所有头寸减半 或 把止损上提到 1×ATR（更紧）
- 突发重大新闻（地缘冲突、政策大变）：立即评估，必要时整体退出
- 流动性枯竭日（节假日前后）：减仓

### 5. 紧急止损（账户级）

| 触发条件 | 动作 |
|---------|------|
| 日内浮亏 -3% | 全平所有头寸 + 当日不再开仓 |
| 单日实现亏损 -3% | 同上 + 第二天降单笔风险到 0.5% |
| 周累计 -6% | 全平 + 周休战 + 周复盘必须找出错的根源 |
| 月累计 -10% | 暂停 1 周 + 重审策略 |
| 单品种持仓亏损达 -3% 账户净值 | 强制止损（不管 ATR 信号） |

## 止损触发后的标准动作

1. **平仓**（市价立即）
2. **记录**到 daily（写明触发原因、是否符合规则）
3. **暂停**该品种 24h——不要立刻"复仇"重新做反向
4. **复盘**：止损是策略错（信号本身错）还是市场噪音？
   - 信号错：调整规则（不是这次单子）
   - 噪音：纪律执行加分，不要懊恼

## 永远不要做的事

❌ **向不利方向移动止损**（"再给它点机会"）——这是炒单巨亏的 #1 原因
❌ **取消止损单**（哪怕"准备重新挂"）——挂的过程会出意外
❌ **倒锤逻辑**：跌到止损了，反而平掉多单转空——除非有新信号确认
❌ **同时给多个头寸"放宽止损"**——风险会同时升高

## 多头止损 vs 空头止损（细节）

### 多头止损（你买涨了）

- 止损 = 开仓价下方
- 工具命令："止损价以下卖平"
- 风险：行情向下加速时，市价单可能比止损价更差（滑点）

### 空头止损（你卖跌了）

- 止损 = 开仓价上方
- 工具命令："止损价以上买平"
- 风险：行情向上加速 + 涨停板 = 出不来

⚠️ 涨停板锁仓是空头最大梦魇。所以：
- 看跌信号弱 + 接近重大利多事件 → 减半空头仓位
- 涨停板上不要硬挂买平单——可能成交价远超止损价

## 止损价位选定的额外考量

ATR 止损是基础，但有几种情况止损位需要调整：

1. **关键技术位**：止损价不要恰好放在前低/前高（容易被插针）。把止损价稍微远一点放在关键位下方/上方更稳
2. **整数关口**：止损不放在 3500、3600 这种整数（容易被止损猎杀）。放在 3493 或 3608 这类不规则数字
3. **跳空缺口**：开盘有跳空时，止损按跳空后价格重新算（不要按昨日收盘）

## 止盈侧的"反向止损"逻辑

止盈本质是反向止损——浮盈达 N×ATR 后启动移动止损保护利润。

为什么这套逻辑能赢：
- 趋势策略胜率 30-40%
- 但盈亏比 3-5:1
- 期望值 = 0.35 × 4 + 0.65 × (-1) = +0.75（每单期望赚 0.75R）

100 次交易期望赚 75R = 75 × 单笔风险（1% 账户） = **75% 账户增长**

但前提是**严格执行**。一次破规可以毁掉 50 次的累积。
