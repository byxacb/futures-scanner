> 这是开赛后**每个交易日**的标准动作清单。**机械执行**——为了让你的注意力留给信号判断，而不是流程记忆。

## 详细动作（按时间）

### ☕ 07:30 起床

1. 给自己 3 分钟清醒——千万不要刚睁眼就看屏幕
2. 看今天天气和心情（自评 1-10）→ 写在 daily/YYYY-MM-DD.md 顶部
3. 心情 ≤5 时降级今日仓位：单笔风险 1% → 0.5%

### 📊 08:00 跑盘前简报

```bash
cd /Users/bianyawen/Desktop/期货
python -m tools.briefing.morning --symbols rb,m,au,IF,sc
```

输出文件：`daily/YYYY-MM-DD.md`

阅读顺序：
1. **第一段：高确信信号** —— 今天有没有？没有就准备空仓
2. 各品种技术面：扫一眼趋势/市态
3. 仓位建议：每个有信号的品种最多开几手

### 🤔 08:15-08:45 决策当日计划

**必答 5 个问题**（写在 daily 顶部）：

1. **今天有没有重大事件？**（FOMC/CPI/钢联/政策会议）
   - 有 → 现有持仓减半，不开新单
   - 没有 → 进入第 2 问

2. **简报里有高确信信号吗？**
   - 没有 → 今天默认空仓
   - 有 1 个 → 标准 1 单位仓位
   - 有 2-3 个 → 检查相关性，避免单方向 >40%

3. **现有持仓的止损还在原位吗？**
   - 检查每笔——浮盈 >1×ATR 的应该已经上移
   - 时间止损：持仓 5 个交易日仍未浮盈 1×ATR → 计划平仓

4. **账户当前回撤是多少？**
   - <3% → 正常 1% 风险率
   - 3-6% → 降到 0.5%
   - >6% → 全平+停手一周

5. **昨日复盘的"明日预案"还有效吗？**
   - 看昨晚 evening.py 输出的明日预案，是否有变化

### 📋 08:50 写当日交易计划

在 daily 文件里加一段：

```markdown
## 今日交易计划

### 现有持仓
- rb2510 多 5 手 @3480 → 止损上移到 3520（成本上方 40 点）
- 其它：[ ]

### 计划新开
- (无 / 等高确信信号)
- 或：m2509 多 8 手 @3520，止损 3440，目标加仓到 12 手 @3580

### 红线
- 单日浮亏 -3% 全平
- 任何品种保证金占用超 10% 立刻减仓
```

### ⏰ 08:55-09:00 挂单

打开比赛软件：
1. 挂今日预备的限价单
2. 同时挂止损单（条件单）
3. 检查所有现有持仓的止损还在

⚠️ 永远不要"市价开盘"——容易被插针

### 🖥️ 09:00 启动盘中监控

```bash
# 在后台开一个终端跑
python -m tools.monitor.live_watch --interval 30 --duration 21600
```

会在触发关键价位时桌面通知。

### 📚 09:00-11:30 / 13:30-15:00 盘中

**纪律**：
- 简报+计划之外的"灵光一现"——**不交易**
- 看到大波动 → 离开屏幕 10 分钟再决定
- 严禁日内反复进出（手续费 ×2 设计就是为了惩罚这个）

**允许的操作**：
- 系统触发的加仓（浮盈 1×ATR 后金字塔加仓）
- 移动止损（仅向有利方向）
- 紧急减仓（单笔超 -2% 风险或日内 -3%）

**禁止操作**：
- 临场决定开新品种
- 修改昨晚定好的止损
- 看微信群"小道消息"立刻动手

### 💸 14:50 收盘前 10 分钟检查

- 持仓是否要平掉留 T+1？
- 止损位是否需要调整以适应夜盘？
- 重大事件是否在夜盘期间发生？

### 📝 15:00-15:30 跑盘后复盘

```bash
python -m tools.briefing.evening --symbols rb,m,au,IF,sc
```

会生成框架文件，**你必须手动填写**：
1. 今日每笔交易明细（进场理由、止损、是否按计划）
2. 账户状态（净值、回撤、持仓）
3. 心理自评（平静度、纪律性、是否触发红线）
4. **教训/经验**（每笔单写一句话，哪怕"按计划执行"）
5. 明日预案

### 🌙 20:30 夜盘前简报（适用品种：au/ag/cu/al/rb/i/sc/MA/PTA）

```bash
python -m tools.briefing.morning --symbols au,sc,cu --out daily/$(date +%Y-%m-%d)_night.md
```

夜盘是国内期货最容易出大行情的时段（受外盘影响）：
- 看欧美开盘前 30 分钟数据/新闻
- 调整现有持仓的止损到夜盘可接受范围
- 没大事件就别开新单——夜盘流动性差、滑点大

### 😴 23:00（或更晚）睡前最后检查

- 所有持仓都有止损单挂在系统里？✓
- 凌晨 02:30 前有重大数据公布？（FOMC 凌晨 2:00、美 CPI 凌晨 21:30 已过）
- 如果有 → 减半仓位 / 提前平掉

## 月末特殊任务

每月最后一个交易日收盘后：
1. **当月排名核对**：和官方排名比较，校准自估模型
2. **品种池效率分析**：哪个品种贡献最多？哪个一直亏？
3. **策略迭代**：基于一个月数据，是否需要调整 master_playbook 参数？
   - 但 **不允许临时大改**——只有月度才能改策略

## 红线触发后的标准应对

| 触发 | 立即动作 | 后续 |
|------|---------|------|
| 单笔亏损 -2% | 强制止损（系统应自动） | 复盘是策略错还是执行错 |
| 日内浮亏 -3% | 全平所有头寸 | 当日不再开仓 |
| 周亏损 -6% | 全平 + 周休战 | 周复盘 + 降单笔风险到 0.5% |
| 月亏损 -10% | 暂停 1 周 | 重审 master_playbook |
| 任何无止损单 | 立即补止损（30 秒内） | 复盘为什么忘了 |

## 不变量（每天提醒自己）

1. **我不是预测者，是反应者**——信号触发就执行
2. **没有止损 = 没开仓**——挂止损单是开仓动作的一部分
3. **空仓也是仓位**——耐心是免费的最大优势
4. **复盘 > 交易**——一份复盘的价值高于 10 笔不复盘的交易
5. **100 天比赛是马拉松不是冲刺**——保住生存权第一
