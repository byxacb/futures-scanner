# 上证 50 IH（CFFEX 中金所）

> 候补品种池——大蓝筹+金融地产权重股指期货，与 IF 高度相关但弹性偏低，反映"中字头"国企景气。

## 合约规格

| 项目 | 参数 |
|------|------|
| 代码 | IH |
| 交易所 | 中国金融期货交易所 CFFEX |
| 合约乘数 | **300 元/点** |
| 报价单位 | 指数点 |
| 最小变动价位 | 0.2 点（1 手 = 60 元 PnL/跳） |

| 交易时间 | 09:30-11:30 / 13:00-15:00（**无夜盘**） |
| 标的指数 | 上证 50 指数（沪市市值最大 50 只） |

## 价格区间参考（近 3 年）

- 历史高点：~3500 点（2021 年 2 月，茅指数高峰）
- 历史低点：~2100 点（2024 年初）
- 近期波动区间：2300-2900 点
- 典型 ATR(14)：30-50 点
- 单手价值 ~80 万，保证金 ~12 万（**资金占用最大**）

## 基本面驱动

### 标的构成（上证 50 权重）
- **金融**（银行+保险+券商）占 30-35%
- **白酒+消费**（茅五泸+伊利）占 15-20%
- **能源+石化**（中石油/中石化）占 8-12%
- **工业制造**（中字头基建+军工）占 8-10%
- **其他**

### 驱动因素
- **央行流动性**：MLF/LPR/降准是核心（金融股权重大）
- **北向资金**：上证 50 是北向首选标的
- **保险/社保配置**：长线增量资金
- **房地产周期**：通过银行、地产链传导
- **白酒消费景气度**（高端白酒）
- **美元/人民币汇率**：人民币贬值时北向流出
- **国企改革+市值管理**政策

### 与 IF 差异
- IH 比 IF 更"大盘+稳"，波动 0.7-0.85 倍 IF
- IH 反映金融地产，IF 反映宽基
- IH/IF 比值反映大小盘风格切换

## 仓位速查（账户 100 万 + 1% 风险）

假设：价格 2600 点、ATR=40、止损距离 80
- 单手亏损 = 80 × 300 = 24,000 元
- 最大手数 = 10,000 / 24,000 ≈ **0 手**（按 1% 风险无法做）
- 放宽至 1.5% 风险（15000）：依然不到 1 手
- 实际做法：**1 手起步且放宽止损**，或等高确信信号再下手
- 保证金占用 = 1 × 2600×300×15% = 117,000 元 ≈ 11.7% 账户（**单手就接近上限**）
- **提醒**：IH 是合约价值最大品种之一，账户 100 万只能 1-2 手

## 历史教训

1. **2021 年 2 月**：核心资产泡沫+北向加速，IH 冲 3500。**经验**：流动性宽松+外资共振时大胆
2. **2022 年**：地产暴跌+俄乌+疫情，IH 从 3300 跌至 2400。**经验**：金融股弱势时 IH 拖累更大
3. **2024 年 9 月底**：政策大转向"924"行情，IH 一周涨 25%。**经验**：政策大底信号必须跟
4. **2024 年 11 月**：政策预期消化，IH 回吐 50% 涨幅。**经验**：政策行情第一波大、第二波难

## 注意事项

- **无夜盘**：周末/隔夜消息次日跳空风险（与 cu/au 不同，IH 不能盘后调整）
- **单手保证金最大**：账户 100 万仅 1-2 手，错一次很伤
- **交割日**第三个周五，临近时基差波动加剧
- **比赛 +3% 保证金**后单手 12 万，账户 100 万持 2 手已用 24%（超单品种 10% 上限），实际只能 1 手
- 政策市特征明显，技术分析作用打折
- 与 IF 相关 0.95+，**不可同时双倍持仓**
