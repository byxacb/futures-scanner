# 豆油 y（DCE 大商所）

> 候补品种池——三大油脂之一，跟随美豆+棕榈油，与 m（豆粕）来自同一压榨链。

## 合约规格

| 项目 | 参数 |
|------|------|
| 代码 | y |
| 交易所 | 大连商品交易所 DCE |
| 合约乘数 | **10 吨/手** |
| 报价单位 | 元/吨 |
| 最小变动价位 | 2 元/吨（1 手 = 20 元 PnL/跳） |
| 涨跌停板 | ±5% |
| 保证金率（交易所） | ~9%；**比赛 +3% ≈ 12%** |
| 主力合约月份 | 1、5、9 月（最活跃通常 1 月或 5 月） |
| 交易时间 | 09:00-11:30 / 13:30-15:00 / 21:00-23:00 |

## 价格区间参考（近 3 年）

- 历史高点：~14000 元/吨（2022 年俄乌+棕榈油暴涨）
- 历史低点：~7000 元/吨（2023-2024 年油脂回归）
- 近期波动区间：7500-9500 元/吨
- 典型 ATR(14)：80-150 元/吨
- 单手价值 = 价格×10，~8 万元；保证金占用 ~1 万

## 基本面驱动

### 供应端
- **大豆压榨量**（钢联+我的农产品网每周数据）
  - 压榨 = 18% 出豆油 + 79% 出豆粕（按重量）
- **美豆产量**（USDA 月度供需 WASDE 报告，每月 12 日附近 24:00 北京时间）
- **巴西/阿根廷大豆**（南美产季 2-5 月）
- **进口大豆到港量**（海关每月）

### 需求端
- **食用油消费**（春节、中秋备货旺季）
- **餐饮业景气度**
- **生物柴油**（占比小但边际有影响，美国/欧洲政策）
- **替代关系**：豆油 vs 棕榈油 vs 菜油（三油价差套利）

### 库存
- **油厂豆油库存**（每周二/五数据）
  - 库存 < 80 万吨 = 偏低看涨
  - 库存 > 120 万吨 = 偏高看跌

### 油粕比
- y / m ≈ 2.0-2.8 常态
- 油粕比 > 2.8 = 油强粕弱
- 油粕比 < 2.0 = 粕强油弱

## 与其他品种的相关性

| 品种 | 相关性 | 说明 |
|------|--------|------|
| 棕榈油 p | 0.85-0.95 | 几乎同步，三油价差套利 |
| 菜油 OI | 0.75-0.90 | 三油另一员 |
| 豆粕 m | 0.40-0.65 | 同压榨链，但油粕比常反向 |
| 美豆 ZS（CBOT） | 0.70-0.85 | 上游 |
| 原油 sc | 0.30-0.50 | 生物柴油渠道 |
| 大商所豆一 a | 0.50-0.70 | 国产大豆 |

**策略含义**：y+p+OI 同向算同一头寸（油脂板块）。

## 仓位速查（账户 100 万 + 1% 风险）

假设：价格 8500、ATR=120、止损距离 240
- 单手亏损 = 240×10 = 2,400 元
- 最大手数 = 10,000 / 2,400 ≈ **4 手**
- 保证金占用 = 4 × 8500×10×12% = 40,800 元 ≈ 4% 账户
- 单品种 10% 上限对应 ~10 手

## 历史教训

1. **2022 年 3 月**：俄乌冲突+棕榈油出口管制，y 从 9000 暴涨至 14000。**经验**：油脂受地缘冲击巨大，敢追也要敢撤
2. **2023 年**：阿根廷干旱推升美豆，但国内油脂高库存压制 y 反弹。**经验**：内外盘可能背离
3. **2024 年**：USDA 报告利空+棕榈油高位回落，y 持续阴跌 4 个月。**经验**:跟随油脂板块大趋势

## 注意事项

- **夜盘 USDA 报告日**（每月 12 日左右 24:00）开仓需减半或离场
- y 流动性强于 OI 弱于 m，主力月清晰
- 春节前一周减仓，防节后跳空
- 三油板块联动强，**不要同时持有 y+p+OI 同方向重仓**
- 棕榈油主导油脂方向时，y 跟随而非主导
