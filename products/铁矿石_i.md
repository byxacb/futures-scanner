# 铁矿石 i（DCE 大商所）

> 候补品种池——黑色产业链上游，进口依赖度 80%+，全球定价品种，国际化合约。波动比 rb/hc 大。

## 合约规格

| 项目 | 参数 |
|------|------|
| 代码 | i |
| 交易所 | 大连商品交易所 DCE |
| 合约乘数 | **100 吨/手** |
| 报价单位 | 元/吨 |
| 最小变动价位 | 0.5 元/吨（1 手 = 50 元 PnL/跳） |
| 涨跌停板 | ±6%（高于 rb，波动更大） |
| 保证金率（交易所） | ~13%；**比赛 +3% ≈ 16%** |
| 主力合约月份 | 1、5、9 月（最活跃通常是 5 月或 9 月） |
| 交易时间 | 09:00-11:30 / 13:30-15:00 / 21:00-23:00 |
| 特征 | 国际化品种，境外投资者可参与 |

## 价格区间参考（近 3 年）

- 历史高点：~1350 元/吨（2021 年 5 月，对应普氏 230 美元）
- 历史低点：~580 元/吨（2022 年中）
- 近期波动区间：700-1000 元/吨
- 典型 ATR(14)：15-25 元/吨
- 单跳 50 元，单手保证金占用约 700×100×16% = 11,200 元

## 基本面驱动

### 供应端
- **四大矿山发货量**：BHP、Rio Tinto、FMG、Vale，钢联每周一汇总
- **巴西 Vale 季节性**：1-3 月雨季减产，4-7 月恢复
- **澳洲飓风季**：12-3 月可能扰动
- **国产矿**（占 15-20%，环保限产时弹性）

### 需求端
- **铁水产量**（钢联每周四数据，最核心指标）
- **247 家钢厂高炉开工率**
- 钢厂利润决定补库节奏

### 库存
- **45 港铁矿石库存**（钢联每周四）：
  - 港口库存 > 1.5 亿吨 = 高位累库 = 看跌
  - 港口库存 < 1.2 亿吨 = 低位去化 = 看涨
- 钢厂库存（厂内）：钢厂库存天数 < 25 天可能补库

### 价格指数
- **普氏 62% Fe 指数**（Platts 每日，美元/干吨）
- 普氏与连铁有汇率+港杂费换算关系（普氏 × 美元汇率 × 1.13 ≈ 连铁）

## 仓位速查（账户 100 万 + 1% 风险）

假设：价格 800、ATR=20、止损距离 40
- 单手亏损 = 40×100 = 4,000 元
- 最大手数 = 10,000 / 4,000 = **2-3 手**
- 保证金占用 = 3 × 800×100×16% = 38,400 元 ≈ 3.8% 账户
- 单品种 10% 上限对应 ~7-8 手
- **提醒**：i 波动大、合约价值大，单手风险是 rb 的 3-4 倍，新手严格 2 手起步

## 历史教训

1. **2021 年 5 月**：碳中和+复苏，i 从 700 冲到 1350，又被发改委约谈打到 600。**经验**：行政干预品种，敢空不敢死扛
2. **2022 年中**：钢厂利润亏损减产，i 从 950 暴跌至 580。**经验**：钢厂利润是核心
3. **2023 年下半年**：发改委多次约谈打压价格，每次都现急跌。**经验**：政策风向比基本面优先
4. **2024 年初**：港口库存累至 1.5 亿吨高位，i 持续 3 个月下跌。**经验**：库存绝对水平很关键

## 注意事项

- **政策风险极高**：发改委约谈是 i 的最大变量，新闻一出常跌停
- **波动 1.5-2 倍于 rb**，仓位必须缩小
- 夜盘成交活跃，普氏指数公布常引发跳变
- **国际化品种**：境外机构持仓影响盘面
- 春节前后供应（巴西雨季）+ 需求（节后补库）共振，常有大行情
- 单手保证金 1 万+，资金占用敏感
