# 10 年期国债 T（CFFEX 中金所）

> 候补品种池——利率风向标，最稳定品种、低波动+高流动性，与货币政策直接挂钩。

## 合约规格

| 项目 | 参数 |
|------|------|
| 代码 | T |
| 交易所 | 中国金融期货交易所 CFFEX |
| 合约乘数 | **10000 元/点**（每点 1 万元） |
| 报价单位 | 元（如 102.500） |
| 最小变动价位 | 0.005 元（1 手 = 50 元 PnL/跳） |
| 涨跌停板 | ±2%（远低于股指） |
| 保证金率（交易所） | ~2.5%；**比赛 +3% ≈ 5.5%** |
| 主力合约月份 | 3、6、9、12 月（季月） |
| 交易时间 | 09:30-11:30 / 13:00-15:15（**无夜盘**） |
| 标的 | 票面 3% 名义长期国债 |
| 合约面值 | 100 万元 |

## 价格区间参考（近 3 年）

- 历史高点：~107（2024 年下半年降息周期高点）
- 历史低点：~96（2020 年中疫情后通胀预期）
- 近期波动区间：102-106
- 典型 ATR(14)：0.15-0.30 元（即 15-30 个跳）
- 单手价值 ~100 万（约等于面值），保证金 ~5.5 万

## 基本面驱动

### 利率驱动（最核心）
- **央行政策利率**：MLF 利率（每月 15 日）、7 天逆回购 OMO 利率
- **LPR**（每月 20 日 09:15）：1Y/5Y
- **降准**：每次 0.25-0.5 个百分点，释放 5000-10000 亿
- **国债收益率曲线**：10Y CGB 收益率每跳 1bp ≈ T 价格 0.08 元

### 经济基本面
- **CPI/PPI**（每月 9-10 日 09:30）
  - CPI 低 = 通缩压力 = 看涨 T
- **PMI**（每月 1 日/4 日）
  - PMI 弱 = 经济差 = 宽松预期 = T 涨
- **社融+信贷**（每月中旬不定期）
  - 社融弱 = 实体融资差 = T 涨
- **工业增加值**（每月 15 日左右）

### 资金面
- **DR007/R007**（每日，央行公开市场+银行间利率）
  - R007 < OMO = 资金宽松 = T 涨
- **同业存单 NCD 1Y 利率**：观察中期资金价
- **季末/年末**资金面紧张周期

### 政策事件
- **政治局会议+中央经济工作会议**：定调宽松/紧缩
- **国债增发/特别国债**：供给冲击短期看跌
- **美联储政策**：中美利差倒挂影响外资+汇率

## 仓位速查（账户 100 万 + 1% 风险）

假设：价格 104、ATR=0.20、止损距离 0.40
- 单手亏损 = 0.40 × 10000 = 4,000 元
- 最大手数 = 10,000 / 4,000 = **2-3 手**
- 保证金占用 = 3 × 104×10000×5.5% = 17,160 元（**注意：T 保证金是按合约价值×保证金率算**）
  - 修正：保证金 = 104 × 10000 × 5.5% ≈ 57,200 元/手；3 手 ≈ 17 万 = 17% 账户
- **再修正后**：3 手已超单品种 10%，实际只能 1-2 手
- 单品种 10% 上限对应 ~1-2 手
- **核心**：T 看似波动小，但合约面值 100 万、保证金占用大，必须按面值算

## 历史教训

1. **2020 年中**：疫情后宽松+复苏预期摆动，T 从 102 跌到 96 又回 102。**经验**：政策切换点波动剧烈
2. **2022 年 11 月**：理财赎回踩踏，T 一周跌 1.5 元（150 跳），机构爆仓。**经验**：债市流动性危机时跌速比股票还快
3. **2023-2024 年**：长期宽松+经济疲软，T 持续走牛冲 107。**经验**：宽松大趋势中坚决多头
4. **2024 年 4 月**：央行罕见喊话"长端利率太低"，T 单日跌 0.7 元。**经验**：政策口头干预即时反应

## 注意事项

- **波动最小**但合约面值 100 万**最大**，资金占用容易超标
- **无夜盘**：周末/隔夜消息次日跳空（特别是周末降准/降息）
- **政策市极致**：央行的一句话可能逆转方向
- **国债增发**+**MLF 缩量续作**等供给/政策事件常常瞬间反应
- **趋势平滑**：T 走势是六大品种中最连贯的，趋势策略表现稳定
- **比赛保证金 +3% 后单手 5.5 万**，1 手已占 5.5%，2 手即 11% 接近单品种上限
- 价格报价小数点 3 位，注意看错小数点
- **股债跷跷板不稳定**：不能简单用 T 对冲股指
