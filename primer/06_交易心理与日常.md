# 第 6 课：交易心理与日常（决定你能不能拿一等奖的最后一公里）

> 读完这课你要知道：交易日的标准流程、3 大心理陷阱、复盘怎么做、什么时候必须停手。

---

## 1. 一个交易日的标准流程

### 盘前（07:30-08:45）

1. ☕ 起床第一件事：开 `tools/briefing/morning.py` 输出的 daily 简报
2. 看：昨夜外盘动向 + 隔夜重要新闻 + 今日关键数据时点 + 各品种技术信号
3. 在脑子里（或日记里）回答 3 个问题：
   - 今天有没有"高确信信号"？没有 → 默认空仓 / 持有现有头寸
   - 我现有头寸的止损还在原位吗？需要移动止盈吗？
   - 今天有没有重大事件（FOMC、非农、政策）？有 → 减半计划仓位

### 开盘（08:55-09:00）

挂好今日预备的限价单 + 止损单。**不要市价开盘——经常被插针**。

### 盘中（09:00-15:00）

- 不需要全程盯——工具会在触发关键信号时桌面通知
- 不在情绪上做决定。看到行情大波动 → **先离开屏幕 10 分钟**再决策
- 严禁日内反复进出（手续费 ×2 设计就是为了惩罚这个）

### 收盘（15:00-15:30）

跑 `evening.py`，输出当日复盘 + 明日预案。

复盘要诚实回答：
- 今天每笔交易：进场理由是什么？是不是按计划执行？
- 亏损单：是策略错还是执行错？
- 盈利单：是运气还是技术？
- 整体账户回撤是多少？符合预算吗？

### 夜盘前（20:30-21:00）

- 有夜盘品种？看看美盘和外盘开盘前 30 分钟
- 调整止损/止盈单到夜盘价格区间
- 没大事件就别开新单——夜盘流动性差、滑点大

### 晚间（21:00-23:00）

- 夜盘活跃但不必通宵——设好止损去睡
- 21:00 重要数据日（CPI、非农、EIA）后第一波行情会很快，没准备好就别参与

### 周末

- 周日跑周复盘：本周得分自估 vs 估算排名
- 看下周关键事件日历
- 调整品种池：剔除最近不出信号或流动性变差的品种

## 2. 3 大心理陷阱（90% 的亏损源于这 3 个）

### 陷阱 1：报复性交易（revenge trading）

刚亏了 5000 → 急着开下一单"找回来" → 仓位翻倍 → 亏更多 → 继续翻倍 → 账户血崩

**应对**：
- 单日亏损达 -3% 强制全平 + 停手一天（电脑挪开、出门、运动）
- 把"我必须赚回来"的念头换成"明天还有 100 天"

### 陷阱 2：过度交易（overtrading）

无聊 → 看着行情想做点什么 → 强行找信号 → 信号不够格也开仓 → 高频小亏积累

**应对**：
- 主策略明确规定每个品种每周最多 2 单（除非高确信信号）
- 一天 0 单是允许的、甚至常态
- 把"无聊"当成成本——比赛是耐心游戏

### 陷阱 3：止损滑动（moving stops）

价格快到止损了 → "再给它一点机会" → 把止损向不利方向移 → 价格继续走 → 亏损扩大 → 再移 → 直到爆仓

**应对**：
- 止损单挂出后**绝不向不利方向移动**（只能向有利方向移，即移动止盈）
- 这条铁律比所有策略加起来都重要

## 4. 什么时候必须停手（强制规则）

立即停手 1 天：
- 日内浮亏达 -3%
- 连续 3 单止损
- 出现报复性念头自己识别到了

停手 1 周（周末复盘后再决定下周是否恢复）：
- 周累计亏损 >6%
- 月累计亏损 >10%
- 出现"我必须搞个大的"念头

永久退出本届比赛：
- 单日亏损 >10%（说明策略已经失控）
- 出现穿仓苗头（保证金占用 >90%）

这些不是"建议"，是**比赛能否拿名次的最低门槛**。

## 5. 一等奖选手的画像（来自历届期货大赛冠军访谈共性）

- 不追涨杀跌
- 整天只看 3-5 个品种
- 大部分时间空仓
- 严格止损（从不犹豫）
- 关键信号时敢加注但不满仓
- 每天复盘
- **情绪稳定**——不亢奋不沮丧

简而言之：**像个冷静的统计员，不像个 gambler**。

## 自测答案（第 5 课的）

1. **做多螺纹**——库存双降 + 产量升说明需求火爆吸收掉所有供给，这是最强看涨组合。但要等技术面确认（如夜盘开盘 30 分钟后突破日内高点）才入场。
2. **平半仓 + 移动止损到成本线**——FOMC 前历来大波动，沪金可能任何方向。锁住一半利润、剩下一半给行情。
3. **远月升水**意味着市场预期未来现货更贵——可能是供应预期收紧（如限产政策传闻）或需求预期改善。趋势策略可以多头偏好；如果你做空近月+做多远月就是跨期套利。

## 第 6 课自测

1. 你早上看到主策略给出 3 个高确信信号（螺纹做多、豆粕做多、沪金做多），你应该全部 3 个一起开吗？
2. 你今天连续止损 2 单，账户 -1.5%，再开 1 单又止损 -0.6%——按规则你接下来该做什么？
3. 复盘时你发现"亏损单全是因为没等技术信号确认就抢进场"——这属于策略错还是执行错？

## 答案

1. **不是**。3 个看多信号同时开 = 单方向（看涨）总仓位过大，遇到大宏观利空全军覆没。规则：单方向仓位 ≤40%。3 个品种最多分配 30% 总仓位，且要看相关性（螺纹和豆粕相关性低 OK，螺纹和铁矿就是高相关）。
2. 累计 -2.1%，距 -3% 还有 0.9%。**应该停手**——3 单连续止损说明今天市场不在你的"频道"上，不要硬干。直接停手，明天再来。
3. **执行错**——策略是对的（等技术信号确认），但你没执行。改方法：用条件单（程序自动到价才挂单）而不是手动盯盘。
