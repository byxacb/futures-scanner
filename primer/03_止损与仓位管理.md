# 第 3 课：止损与仓位管理（拿一等奖的真正秘密）

> 读完这课你要知道：什么是 ATR、怎么用账户净值反推手数、为什么"满仓"是新手最大杀手、凯利公式简化版。

---

## 1. 仓位管理为什么是命

随便挑一本《Market Wizards》翻开，所有顶级交易员第一句都说**"风险管理 > 选股"**。原因：

- 选股错了，控仓得当 → 小亏退出，下次再来
- 选股对了，仓位过大 → 第一次震荡就被洗出来
- 选股错了 + 仓位过大 → 一单破产

本次比赛评分公式 60% 权重在惩罚风险——意思是仓位管理决定 60% 的分数。

## 2. ATR：决定你该开多少手的核心指标

**ATR = Average True Range**（平均真实波幅），衡量品种最近 N 天（通常 14 天）平均每天波动多大。

举例：
- 螺纹钢 ATR(14) = 50 元/吨——意思是螺纹钢平均每天上下波动 50 元/吨
- 沪铜 ATR(14) = 800 元/吨——铜波动比螺纹大 16 倍

**ATR 越大 = 品种越剧烈 = 你应该开越少手**（不然单日波动就把你账户晃出去）

ATR 计算（每天的 True Range）：
TR = max(今日最高 - 今日最低, |今日最高 - 昨日收盘|, |今日最低 - 昨日收盘|)
然后取最近 14 天 TR 的平均。

工具会算给你，本课只需知道**ATR 越大代表波动越大**。

## 3. 仓位计算公式（背下来，每次交易都要算）

**单笔最大亏损 = 账户净值 × 风险比例**（本次比赛 2%）

**止损距离 = 2 × ATR**

**最大手数 = 单笔最大亏损 / (止损距离 × 合约乘数)**

### 例 1：螺纹钢

- 账户净值 100 万，风险比例 2% → 单笔最大亏损 = **20,000 元**
- 螺纹钢 ATR = 50 元/吨 → 止损距离 = 100 元/吨
- 合约乘数 10 吨/手 → 1 手亏 100 元/吨×10 = 1,000 元
- 最大手数 = 20,000 / 1,000 = **20 手**
- 保证金占用 ≈ 20 × 4,200 = 84,000 元（占账户 8.4%）

### 例 2：沪铜

- 账户净值 100 万，风险比例 2% → 单笔最大亏损 = 20,000 元
- 沪铜 ATR = 800 元/吨 → 止损距离 = 1600 元/吨
- 合约乘数 5 吨/手 → 1 手亏 1600×5 = 8,000 元
- 最大手数 = 20,000 / 8,000 = **2.5 手**（取整 → 2 手）
- 保证金占用 ≈ 2 × (75000×5×12%) = 90,000 元（占账户 9%）

注意：**同样 2% 风险，开沪铜只能开 2 手，开螺纹能开 20 手**——但保证金占用差不多。这就是 ATR 仓位法的妙处：自动让你在波动大的品种少开，波动小的多开。

## 4. 总仓位上限（本次比赛硬规则）

| 维度 | 上限 | 说明 |
|------|------|------|
| 单品种保证金占用 | ≤10% | 单个品种最多用 10 万保证金（按 12% 算约 1 手沪铜或 23 手螺纹） |
| 单方向（多/空合计）保证金占用 | ≤40% | 不能 8 个品种全做多 |
| 总保证金占用 | ≤70% | 留 30% 应对追保和黑天鹅 |
| 单笔亏损 | ≤2% 账户 | 强制止损 |
| 日内总亏损 | ≤3% 账户 | 触发立刻全平、停手一天 |

## 5. 高确信信号的"加注"机制

主策略说"受控激进"——意思是大部分时间稳健（基础仓 30-40%），**少数高确信时刻**敢加注到 60-70%。

什么算高确信？需要至少 3 个独立条件同时成立：
1. ✅ 技术面：突破 20 日通道 + 成交量放大
2. ✅ 基本面：基本面新闻支持方向（如政策、库存、外盘联动）
3. ✅ 情绪面：未到极端（极端时反转概率大）
4. ✅ 多品种联动：板块内其他品种同向（如做多螺纹 + 铁矿、热卷同涨）

3 项以上同时具备 → 可以从 1 单位仓位加到 2-3 单位（金字塔加仓，仅在浮盈 1×ATR 后才加）。

⚠️ **绝不在亏损头寸上加仓**——这是马丁加倍，是破产快车。

## 6. 凯利公式简化版（高级一点的数学）

凯利公式告诉你"在已知胜率和盈亏比的情况下，最优仓位比例是多少"。

**简化版**：f* = (胜率 × 盈亏比 - (1-胜率)) / 盈亏比

举例：胜率 40%，盈亏比 3:1（赢一次赚 3，输一次亏 1）
f* = (0.4 × 3 - 0.6) / 3 = (1.2 - 0.6) / 3 = **20%**

意思：理论上每次该用 20% 资金。但**实际操作要打 1/4 折扣**（凯利分之四），变成 5%——因为真实胜率和盈亏比都有估计误差，全凯利会过度承担风险。

**本次比赛默认**：单笔 2% 是凯利分之四之后的保守值，对应胜率 40-50%、盈亏比 2-3 的趋势策略。

## 7. 资金管理的"3 道防线"

```
第 1 道：单笔止损 2%       ← 每笔交易都要触发
第 2 道：日内亏损 3%       ← 触发后全平+停手一天
第 3 道：周亏损 6%         ← 触发后周末复盘+降仓
```

- 第 1 道失守：日常成本，每周可能踩 2-3 次都正常
- 第 2 道失守：罕见但会发生（比赛 100 天可能 3-5 次）
- 第 3 道失守：严重——意味着策略或心态有大问题，必须停手反省
