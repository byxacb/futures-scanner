# 第 4 课：技术分析入门（看图怎么入手）

> 读完这课你要知道：K 线怎么看、均线/MACD/RSI/布林是什么、什么是趋势/震荡、突破和假突破。

---

## 1. K 线：一根棒子里有 4 个价

每根 K 线对应一个时间周期（5 分钟、15 分钟、1 小时、日、周……）。

```
        ─── 最高价
        │
   ┌──┐
   │  │ ← 收盘价（阳线时这里）
   │实│
   │体│
   │  │ ← 开盘价（阳线时这里）
   └──┘
        │
        ─── 最低价
```

- **阳线**（红色或空心）：收盘价 > 开盘价（涨）
- **阴线**（绿色或实心）：收盘价 < 开盘价（跌）
- **影线**：上下两端的细线，代表最高/最低被打到过但收回来了
- **十字星**：开盘 ≈ 收盘——多空胶着，是潜在反转信号

经典 K 线形态（看图找）：
- **吞没**：今天大阳线完全包住昨天阴线 → 反转看涨
- **锤子线**：长下影、短实体 → 跌势末端反转看涨
- **黄昏星**：连续三根 = 大阳 + 十字 + 大阴 → 涨势末端反转看跌
- **缺口**：开盘价跳空高于昨日最高（或低于昨日最低）→ 跳空多半会回补

⚠️ 单根 K 线**不能孤立解读**——必须放在趋势上下文里看。

## 2. 时间周期的层级

期货分析至少看 3 个周期：
- **大周期**：周线 / 日线 → 判断整体趋势（多/空/震荡）
- **中周期**：4 小时 / 1 小时 → 寻找进场窗口
- **小周期**：15 分钟 / 5 分钟 → 精细化进场时机

**铁律：大周期定方向，小周期找点位**。日线下跌趋势里去做 5 分钟做多 = 逆水行舟。

## 3. 均线（MA / Moving Average）

最简单也最有用。MA(N) = 最近 N 根 K 线收盘价的平均值。

常用配置：
- **MA5 / MA10**：短期，波动大
- **MA20**：中期（约一个月），最常用
- **MA60**：长期（季度），代表趋势线
- **MA120 / MA250**：大趋势

### 看均线的 3 个用法

1. **方向**：MA20 向上 = 上升趋势；MA20 向下 = 下降趋势；MA20 走平 = 震荡市
2. **支撑/阻力**：价格回踩 MA20 不破 → 强势上升趋势继续；跌破 MA20 → 趋势可能转弱
3. **金叉死叉**：MA20 上穿 MA60 = 金叉（看多）；MA20 下穿 MA60 = 死叉（看空）

**比赛主策略以均线为基础信号**：日线 20/60 金叉做多、死叉做空（仅当 ATR 突破确认）。

## 4. MACD（指数平滑异同移动平均）

进阶版均线，由两条线 + 一个柱状图组成。
- **DIF（快线）**：EMA(12) - EMA(26)
- **DEA（慢线）**：DIF 的 EMA(9)
- **MACD 柱**：(DIF - DEA) × 2

用法：
- **DIF 上穿 DEA**：金叉看多
- **DIF 下穿 DEA**：死叉看空
- **零轴上**：多头市场；**零轴下**：空头市场
- **背离**：价格创新高但 MACD 没创新高（顶背离）→ 趋势衰竭信号

MACD 的优点：把均线交叉提前并平滑化。缺点：滞后，震荡市频繁误信号。

## 5. RSI（相对强弱指标）

衡量"涨/跌"的强度，0-100。

- RSI > 70：超买（涨过头，可能回调）
- RSI < 30：超卖（跌过头，可能反弹）
- RSI 在 40-60：震荡区
- **背离**比绝对值更重要：价格新高 RSI 没新高 → 顶部信号

⚠️ 强趋势里 RSI 可以长时间 >70 或 <30（连续暴涨/暴跌），不要硬反着做。

## 6. 布林带（Bollinger Bands）

由 3 条线组成：
- **中轨**：MA20
- **上轨**：MA20 + 2×标准差
- **下轨**：MA20 - 2×标准差

价格 95% 时间在带内，5% 时间在带外。

用法：
- **触上轨**：超买；**触下轨**：超卖（在震荡市）
- **带宽收窄**：波动率压缩 → 即将爆发大行情（多空都可能）
- **带宽急扩**：趋势开启，沿着上/下轨走
- **趋势市突破布林后回踩中轨不破**：趋势延续

## 7. 趋势 vs 震荡：你必须先判断这个

策略选择完全取决于市场状态：

| 市场状态 | 特征 | 用什么策略 |
|---------|------|----------|
| **趋势市** | MA 持续向上/向下、价格创新高/新低 | 趋势跟踪（突破做多/空、移动止盈） |
| **震荡市** | MA 走平、价格在区间内反复 | 高抛低吸或观望（趋势策略易亏） |
| **加速市** | MACD 柱拉长、价格急涨/急跌 | 减仓兑现 + 等回调 |

判断方法（简化）：
- ADX 指标 > 25 → 趋势市
- ADX 指标 < 20 → 震荡市

**比赛主策略偏向趋势市**——震荡市的最优做法是少做或不做。

## 8. 突破与假突破

突破：价格冲出关键价位（前高、前低、布林上下轨、整数关口）。

**真突破特征**：
- 成交量明显放大（>20 日均量 1.5 倍）
- 突破后回踩不破突破位
- 同板块其他品种同步发动

**假突破特征**：
- 突破后立刻收回（"插针"行情）
- 量价背离（突破时量没放大）
- 突破在重要时点附近（夜盘开盘 30 分钟内、收盘前 30 分钟）

**比赛原则：宁可漏单不要追单**——评分公式偏爱低波动，假突破的回撤代价比错过一次正确突破大。

## 9. 持仓量（OI）和成交量

期货独有指标：

- **成交量（Volume）**：今天买卖多少手
- **持仓量（OI / Open Interest）**：还没平的合约数

变化解读：
- 涨 + 量增 + 持仓增 → **多头强势开仓**，趋势可能延续
- 涨 + 量增 + 持仓减 → 空头止损，可能末端
- 跌 + 量增 + 持仓增 → 空头强势开仓
- 跌 + 量增 + 持仓减 → 多头止损

**比赛工具会自动算这些**——你只需要看每天简报里的"多空主力博弈"那一栏。
